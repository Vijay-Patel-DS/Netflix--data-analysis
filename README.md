# Netflix Data Analysis

This project performs Exploratory Data Analysis (EDA) on a Netflix dataset using Python.

## Objective
- Analyze Movies vs TV Shows
- Identify top genres and countries
- Study content trends over time

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Insights
- Movies dominate Netflix content
- Certain genres are more frequent
- Content increased over years
